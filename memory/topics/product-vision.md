# 产品起源与愿景

## 起源故事 (2025年8月)
- **痛点**: 互联网投资内容混乱，新手难以辨别信息质量，传统教育枯燥
- **洞察**: 交易直觉（盘感）是一种可训练的能力，但缺少零门槛的训练工具
- **灵感来源**:
  - Tinder — 滑动交互的直觉映射（右=喜欢=买，左=不喜欢=卖）
  - TikTok — 无限流体验，停不下来
  - Duolingo — 游戏化留存（连续打卡/等级/排行榜）
  - 跳一跳 — 微信生态内的轻量游戏（零安装，社交裂变）

## 目标用户
- **年龄**: 18-35岁
- **特征**: 对投资好奇但无实操经验，碎片化时间，习惯手机操作
- **场景**: 通勤/午休/排队，每次3-5分钟
- **需求**: 想"试试看"投资是什么感觉，但不想承担真实风险

## 产品定位
- **核心**: 投资直觉测试游戏，NOT 专业投资教育
- **体验**: 滑动决策(3秒) → 即时K线反馈 → AI复盘分析
- **合规**: 不涉及真实交易，虚拟金币制，定位"直觉测试"非"投资建议"
- **商业化方向**: 免费游戏 + 高级AI分析/历史回看/自选股付费

## 关键转折（产品迭代中的重大方向调整）

### 1. 教育优先 → 娱乐优先
- **原始想法**: 教用户看K线、理解MACD等指标
- **现实**: 用户会跳过所有教学模块，直接想玩
- **结论**: 先让人玩起来，教育在复盘中自然渗透

### 2. 复杂UI → 极简
- **原始想法**: K线图上叠加均线、MACD、RSI等技术指标
- **现实**: 信息过载让新手恐惧，与"零门槛"理念矛盾
- **结论**: 只留K线+滑动，指标未来作为高级功能

### 3. 弹窗结果 → 独立结算页
- **原始想法**: 交易完成后原地弹窗显示结果
- **现实**: 弹窗打断体验，不够专业，信息展示空间有限
- **结论**: 独立 result 页面，完整展示战绩+K线回顾+AI分析

### 4. 单一指标 → 多维反馈
- **原始想法**: 只看总收益率
- **现实**: 单一数字无法帮用户理解自己的交易行为
- **结论**: 胜率 + 回报率 + 交易标记(买卖点) + AI分析

### 5. 孤立K线段 → 连续可回溯图表
- **原始想法**: 每个回合显示独立的K线片段
- **现实**: 割裂感强，用户无法感受到"一段行情"的完整故事
- **结论**: 连续K线序列，可水平回溯查看历史

### 6. 赌博感 → 金融软件审美
- **原始想法**: 游戏化=花哨（鲜艳颜色/大字号/多动效）
- **现实**: 太像赌博app会引发合规问题和用户不信任
- **结论**: 深蓝+金色的专业金融软件审美，但保留游戏化交互

## 竞争优势
- **零门槛交互**: 滑动即决策，不需要任何金融知识
- **真实历史数据**: 58只真实股票的历史K线，非虚构数据
- **AI分析复盘**: DeepSeek 深度分析 + 本地规则即时反馈
- **社交裂变**: 微信生态原生，分享战绩/挑战好友/排行榜
