# 用户版核心产品介绍（App Store / 小程序简介）

**产品名**：盘感 Trading Instinct

**一句话简介**
用小游戏训练你的投资直觉 —— 左滑看跌，右滑看涨，真实历史行情切片，AI一句话讲明白对错原因。

**功能亮点**
- 立即上手：不需要金融基础，滑动即可开始
- 真实历史：来自真实股票的历史区间，回合结束揭晓标的与时间
- 快速反馈：每局 5–10 次决策，结算即看收益
- AI点评：基于指标的简单解释（MA/RSI/成交量等）
- 成长系统：段位、连胜奖励、错因报告

**首批种子关卡（示例）**
- 新手关卡：Apple、Google 稳定上涨期（建立信心）
- 经典战役：2008 金融危机、2020 疫情冲击期
- 传奇时刻：Tesla 翻百倍、比特币牛市（后期可选）
- 中国特色：茅台长牛、新能源爆发

**候选 Slogan（用户向）**
- 左滑右滑，盘感自来
- 像刷抖音一样学投资
- 你的投资直觉值多少钱？

**隐私与合规提醒**
本应用仅为**模拟教育游戏**，不含真实投资、无资金交易功能。所有提示仅供学习参考，不构成任何投资建议。
