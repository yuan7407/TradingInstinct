# 结果页模板（可直接复制到小程序界面）

以下为一份简洁的本轮结束页模板（文字版与 ASCII 视觉版）。UI 文案、按钮与分享点都在下方建议里。

## ASCII 视觉版
┌─────────────────────────────┐
│        本轮战绩              │
├─────────────────────────────┤
│   恭喜！你赚了 326%          │
│                             │
│   这是：英伟达 NVDA          │
│   时间：2023.01 - 2024.01    │
│                             │
│   如果投入1万元              │
│   现在价值：42,600元         │
│                             │
│   击败了全球78%的玩家        │
│                             │
│ [分享战绩] [再来一局] [查看详情]│
└─────────────────────────────┘

## 结果页要素（可编辑）
- 摘要成绩（百分比）
- 标的 + 时间区间（年/月）
- 货币化示例（如果投入 X 元，现在价值 Y 元）用于提升代入感
- 排名/击败比例（社交驱动力）
- 三个 CTA：分享 / 再来一局 / 查看详情（查看详情展开技术指标与AI点评）

## 分享文案示例
- 我在盘感里用 1 万打了 NVDA（2023）赚了 326%！看看你能不能超越我〜🔁
- 分享图片包含：头像、段位、%收益、标的与年份（不泄露具体日期）
