# 产品原型（低保真，可编辑）

## 核心理念
盘感（Trading Instinct）是一款把历史K线切片“游戏化”的小程序：用户通过左右滑动（看涨/看跌）来预测接下来一段时间的价格走向，回合揭晓后给予虚拟金币奖励并由AI基于后端计算的技术指标做简短点评。

## 核心流程（屏幕与交互）
1. 启动 / 首页
   - CTA：立即开始（匿名试玩）
   - 今日挑战 / 排行榜入口 / 个人成长徽章

2. 游戏主界面（回合内）
   - 屏幕上方：被遮挡的 K 线图（仅显示到 T）；图像可拖拽缩放（简化版不必实现）
   - 中央：滑动区域（左滑=看跌、右滑=看涨）
   - 底部：当前局数 / 已玩轮数 / 信心值选择（×1/×2/×3，可选）
   - 小角落：提示 “非投资建议 / 模拟教育游戏”

3. 结算（单次回合内即时揭晓）
   - 展示本轮立即收益、点击查看指标、AI一句点评（120字内）、继续下一轮按钮

4. 本轮结束页（示例见文件 `07_result_page_template.md`）

5. 个人页
   - 段位、错因报告（每周一份）、历史战绩、分享记录

## 交互与时序
- 单回合：5–10 次左右滑动为一轮
- 每次滑动触发：
  1. 前端记录用户选择（方向 + 信心）
  2. 后端返回未来 N 根（已预生成）并计算分数
  3. 前端展示分数、动画（金币飞入）并触发 Claude 解释（短文本）

## 原型草图（文字）
```
[首页] -> [开始游戏] -> [被遮挡K线图]
    用户：左滑/右滑（可选信心倍数）
    -> 展示下一段K线 + 今日收益更新
    -> 重复 5-10 次 -> 本轮结算页面 -> 自动开始新轮或返回首页
```

## 设计注记（MVP）
- 初版只做日线切片（稳定、易授权）
- 切片池按“牛/熊/震荡”均衡分配
- 所有解释数据由后端计算，AI 仅做“通俗化表达”
- 重点在“快感循环”与“简单分享”机制

## 用户旅程示意（简短）
1. 新手：第一轮内展示 Apple/Google 稳定上涨的切片 → 建立信心
2. 进阶：引入 2008 / 2020 等经典战役切片 → 强化历史认知
3. 高阶：传奇时刻（Tesla 百倍）/ 中国特色（茅台、新能源爆发）
