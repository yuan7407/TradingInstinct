# Claude 使用模板与示例（请务必让后端先计算指标再调用 Claude）

## 标准输入 JSON（示例）
```json
{
  "symbol":"NVDA",
  "period":"2023-01-01_2024-01-01",
  "lastVisibleIndex": "2023-06-30",
  "indicators": {
    "MA5": 230.4,
    "MA20": 210.1,
    "RSI14": 68.2,
    "KDJ": {"K": 87, "D": 79, "J": 95},
    "volume": 1200000,
    "volume_ma20": 800000
  },
  "actual_outcome": "up",
  "user_guess": "up",
  "user_confidence": 2
}
```

## Claude Prompt 模板（中文，示例）
```
你是一个简洁可信的技术面解释器。输入是结构化 JSON（真实指标数值已经计算好）。
请用不超过120字的中文完成两件事：
1) 简短说明当时哪些指标支持或反对用户的判断（列出关键指标和数值）。
2) 给出一条具体可执行的改进建议（例如：下次注意 MA 均线金叉 / 留意成交量放大）。
语气亲切、简短、面向新手但不低俗。不要编造数值或时间。
```

## 示例输出（示例）
> 当时 MA5 在 MA20 之上（230 vs 210），RSI 接近 70，量能放大，支持看涨。但 KDJ 已超买，说明短线回调风险高。建议：若信心不高可等回踩到 MA5 附近再加仓。
